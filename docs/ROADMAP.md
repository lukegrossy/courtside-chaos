ROADMAP

Vision

Create the world’s most entertaining minor-league basketball story generator.

Not the deepest.

Not the most realistic.

The most memorable.

Success is measured by players saying:

“You’ll never believe what happened this season.”

⸻

Development Philosophy

Build stories first.

Systems second.

Polish third.

If a feature does not create stories, remove it.

⸻

PHASE 1

Vertical Slice

Goal:

Prove the core loop is fun.

Player experiences one week of franchise ownership.

Features

* Sandra briefing
* Random event
* Player decision
* Outcome
* Newspaper headline
* Franchise history log
* Next week button

Success Criteria

Player wants to click:

One More Week

⸻

PHASE 2

Season Prototype

Goal:

Complete a full season.

Features

* 30+ events
* Character memories
* Team reputation
* Fan support
* Cash balance
* Weekly game results
* Season standings (lightweight)
* End-of-season newspaper

Success Criteria

Players tell stories from their season.

⸻

PHASE 3

Character Expansion

Goal:

Make characters feel alive.

Features

* Sandra memory system
* Ricky story arcs
* Tiny story arcs
* Coach Hank story arcs
* Commissioner investigations
* Rival team personalities

Success Criteria

Players develop favourites.

⸻

PHASE 4

Franchise History

Goal:

Create unique franchise narratives.

Features

* Newspaper archive
* Season records
* Hall of Fame
* Retired legends
* Historic disasters
* Historic championships

Success Criteria

Every franchise feels different.

⸻

PHASE 5

Promotion Race

Goal:

Give players something to chase.

Features

* League reputation
* Association attention
* Scout appearances
* Promotion requirements
* League awards

Success Criteria

Promotion feels earned.

⸻

PHASE 6

Content Explosion

Goal:

Massive replayability.

Features

* 500+ events
* 100+ mascot disasters
* 50+ sponsor events
* 50+ rivalry events
* 50+ fan events
* Dynamic event chains

Success Criteria

No two seasons feel the same.

⸻

PHASE 7

Presentation Upgrade

Goal:

Become visually distinctive.

Features

* Illustrated character portraits
* Newspaper layouts
* Arena artwork
* Team branding
* Audio system
* VHS-inspired presentation

Success Criteria

Screenshots instantly recognizable.

⸻

Future Features

Potential additions:

* Team relocation
* Expansion franchises
* Franchise owner rivals
* Local television deals
* Documentary mode
* Hall of Fame ceremonies
* Dynasty mode

Only if they create stories.

⸻

Explicit Non-Goals

The game will NOT focus on:

* advanced statistics
* salary cap simulation
* realistic contracts
* detailed tactics
* playbook management
* player attribute spreadsheets

These systems create work.

Not stories.

⸻

Current Milestone

M1 - First Playable Week

Required Screens:

1. Title Screen
2. Sandra Briefing
3. Event Choice
4. Outcome
5. Newspaper
6. History Log

When this works:

Ship a browser prototype.

Get feedback.

Continue.

⸻

The Golden Rule

Every feature must answer:

Will somebody tell a friend about this?

If the answer is no:

Cut it.
